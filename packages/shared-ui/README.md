# @pls/shared-ui

Reusable component library for consistent UI across all lenses and the workspace shell.

## Components

| Component | Description |
|-----------|-------------|
| `Badge` | Status indicator badges |
| `Button` / `IconButton` | Icon-based action buttons |
| `Dialog` | Modal dialogs |
| `EmptyState` | Placeholder for empty data states |
| `ProgressBar` | Progress visualisation |
| `Spinner` | Loading indicator |
| `StatusDot` | Small coloured status indicators |
| `Switch` | Toggle switches |
| `TagPill` | Styled tag display |
| `SectionLabel` | Section headers |

## Utilities

| Export | Description |
|--------|-------------|
| `cn()` | Classname merging (clsx + tailwind-merge) |
| `TAG_STYLES` | Colour tokens for tag types |
| `COURSE_COLORS` | Colour tokens for courses |

## Dependencies

clsx, tailwind-merge, class-variance-authority, lucide-react
