# @pls/lens-test-me

Self-testing lens with spaced repetition and confidence tracking.

## Features

- Quiz interface with reveal-answer pattern
- Prioritises confused segments and low-confidence topics
- Exam mode with countdown timer
- Difficulty indicators (easy, medium, hard)
- Confidence visualisation (0–100%)
- "Got it" / "Didn't get it" feedback updates confidence scores

## Scope

Uses tagged segments to prioritise which questions to surface.

## Internal Dependencies

`lens-framework`, `substrate`, `shared-ui`
